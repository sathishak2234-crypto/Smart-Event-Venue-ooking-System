# Cloud File Security System using Face Authentication

This is a sample project for juniors demonstrating:
- Deep Learning based face authentication
- Secure file upload
- File encryption before cloud storage

## Technologies Used
- Python
- Flask
- Face Recognition (CNN based)
- AES Encryption

## How to Run
1. Install requirements  
   pip install -r requirements.txt

2. Run app  
   python app.py

3. Upload face image and file

## Note
This is a demo version for academic learning.
